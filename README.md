# SCOOT

The code for the oral paper  "**SCOOT: SLO-Oriented Performance Tuning for LLM Inference Engines**"  at The ACM Web Conference 2025 (WWW'25) has passed the open-source review of Ant Group. 

The code repository address is: https://github.com/Ketonmi/SCOOT-vLLM. Here, we have open-sourced the code that can be used to optimize vLLM-0.5.5. 

If you have any questions about the code implementation, please directly raise an issue in the code repository or contact the first author of the paper via email at ketonmi@outlook.com.
